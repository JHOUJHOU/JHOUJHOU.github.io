情境四：預算內的商品

客人：「想找 4 萬以下，而且有現貨的沙發」
小美想查：要同時符合：是沙發、4萬以下、有庫存

情境問題拆解：

  • 使用 SELECT 查詢語法，查詢 商品名稱、商品價格、商品庫存、商品分類 欄位。
  • 使用 WHERE 篩選條件語法 及 AND 且 語法，篩選同為
    • 沙發 類的商品
    • 運用 比較運算子 < 小於 語法 ，篩選出價格在 4 萬元以下的沙發商品
    • 運用 比較運算子 > 大於 語法，篩選出沙發庫存大於 0 的沙發商品

情境五：特價且有貨

客人：「沙發有哪些特價且現貨的品項？」
小美想查：要找到沙發類且有特價（原價大於優惠價）且還有庫存的商品

情境問題拆解：

  • 使用 SELECT 查詢語法，查詢所有欄位。
  • 使用 WHERE 篩選條件語法 及 AND 且 語法，篩選同為
    • 沙發 類的商品。
    • 運用 比較運算子 > 大於 語法，篩選出沙發類的商品中，原價大於優惠價的商品。
    • 運用 比較運算子 > 大於 語法，篩選出原價大於優惠價的商品並且有庫存的沙發類商品。

情境七：指定商品

客人：「北歐風雙人沙發和貓抓皮L型沙發哪個還有貨？」
小美想查：要找出這兩張特定沙發的庫存狀況

情境問題拆解：

  • 使用 SELECT 查詢語法，查詢所有欄位。
  • 使用 WHERE 篩選條件語法 、 AND 且 、OR 或 語法，篩選
    • 運用 比較運算子 = 等於 語法，篩選出 北歐風雙人沙發 或 貓抓皮L型沙發 的商品。
    • 運用 比較運算子 > 大於 語法，篩選出 北歐風雙人沙發 或 貓抓皮L型沙發 商品庫存大於 0 的沙發商品
